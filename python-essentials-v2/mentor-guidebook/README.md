# Python Essentials Mentor Guidebook

Thanks for mentoring in CTD's Python class! Use this folder to access resources for mentoring sessions.

## Folder Structure

- `group-lesson-plans/` - Weekly lesson plans for group mentoring sessions
- `assignment-solution-examples/` - Sample solutions and rubrics for each lesson

## Curriculum Schedule (Aligned with v1)

| Week | Topic | Key Concepts |
|------|-------|--------------|
| 1 | Introduction to Python | Variables, operators, control flow, functions |
| 2 | Data Structures & File Handling | Lists, tuples, sets, dictionaries, file I/O |
| 3 | More Python Skills | Functions, conditionals, input handling |
| 4 | Intro to Data Engineering | Pandas basics, Series, DataFrames, CSV/JSON |
| 5 | Data Wrangling & Aggregation | Selection, groupby, merge, pivot tables |
| 6 | Data Cleaning & Validation | Missing data, duplicates, outliers, apply() |
| 7 | Advanced Data Cleaning | Regular expressions, fuzzy matching, transform |
| 8 | Databases & SQL | SQLite, table relationships, JOINs, Pandas integration |
| 9 | Introduction to Machine Learning | Scikit-Learn, regression, classification, evaluation |
| 10 | Flask Web Development | Routes, templates, forms, REST APIs |
| 11 | Deployment & Final Projects | Cloud hosting, CI/CD, project showcase |

## Session Formats

**Explore Sessions** → Walk through new concepts with live coding demonstrations  
**Apply Sessions** → Debug student code, work through challenges, troubleshoot issues
